
Как это работает:
1. Читаем файл построчно.
2. Разбиваем строки на слова, убирая знаки препинания.
3. Используем `Map` для подсчета частоты слов.
4. Сортируем слова по убыванию их частоты.
5. Выводим топ-10 слов.

Требования:
- Файл `file.txt` должен существовать и содержать текст.
- Код учитывает слова без учета регистра (`toLowerCase`).

 Ключевые моменты:
1. Обобщенный массив: Для создания массива используется `@SuppressWarnings("unchecked")` и приведение к типу `(T[]) new Object[capacity]`.
2. Методы:
   - `push`: Добавляет элемент в стек, проверяя переполнение.
   - `pop`: Удаляет верхний элемент и возвращает его.
   - `peek`: Возвращает верхний элемент без удаления.
3. Обработка ошибок: Исключения выбрасываются, если стек пуст или переполнен.


 Объяснение:
1. Класс Product:
   - Реализует Comparable для сортировки по названию.
   - Содержит цену, количество и вычисляет общую стоимость товара.

2. TreeSet:
   - Сортирует товары автоматически по названию.

3. Map для популярности:
   - Учитывает количество проданных единиц каждого товара.

4. Функционал:
   - addSale: Добавляет товар и учитывает его популярность.
   - listSales: Выводит список товаров.
   - getTotalSales: Подсчитывает общую сумму продаж.
   - getMostPopularProduct: Находит самый популярный товар.
